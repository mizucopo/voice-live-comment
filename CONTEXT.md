# Voice Live Comment

Voice Live Comment turns spoken words into YouTube Live comments.

## Language

**STTプロバイダー**:
音声をテキスト化するために利用する認識サービスまたはブラウザ機能。
_Avoid_: 音声認識エンジン

**VAD**:
音声入力の中でコメント候補にする区間の開始と終了を判定する発話区間判定。
_Avoid_: 音声認識, 文字起こし

**音声コメントセッション**:
ユーザーの開始/停止操作から、STTプロバイダーの開始、外部音声パイプラインの制御、認識結果の投稿、状態通知までを含む一回の音声コメント実行単位。
_Avoid_: 音声認識セッション

**発話冒頭**:
ユーザーが一つのコメントとして話す内容の最初の部分。
_Avoid_: 話し始め, 頭

**冒頭欠け**:
発話冒頭が音声入力または文字起こしに含まれず、コメント文の先頭が抜けた状態。
_Avoid_: 頭切れ, 先頭欠落

**発話前音声**:
VADが発話開始と判定する前に録られていたが、同じコメントに属しうる音声。
_Avoid_: 先読み音声, pre-roll

**ワードブースト**:
コメントとして認識されやすくしたい固有名詞や定型句。
_Avoid_: keyterm, 認識ヒント

**発話終了猶予**:
VADが短い無音を同じコメント候補に含めるために、発話終了の確定を待つ猶予。
_Avoid_: 間, silence timeout, 無音タイムアウト

**コメント混入**:
前のコメントに属する音声が、次のコメントの文字起こし候補に含まれてしまう状態。
_Avoid_: 混ざり, 重複

**配布版**:
利用者がChromeに追加するために提供されるビルド済みのVoice Live Commentパッケージ。
_Avoid_: ビルド成果物, アーティファクト

**配布版インストール**:
利用者がビルド済みの拡張機能パッケージを使ってVoice Live CommentをChromeに追加すること。
_Avoid_: 開発環境セットアップ

**リリースタグ**:
配布版のバージョンを一意に表す `vX.Y.Z` 形式の識別子。
_Avoid_: git tag, タグ
